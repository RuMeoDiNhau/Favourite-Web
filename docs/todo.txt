KẾ HOẠCH TRIỂN KHAI ĐỀ TÀI 3: APPLICATION DEVELOPMENT ON AWS (WEB ON CLOUD)
Dự án: Fav Web (Face Recognition Portal - Games, Music, Knowledge)
================================================================================

KIẾN TRÚC MỤC TIÊU TRÊN AWS (TARGET ARCHITECTURE)
--------------------------------------------------------------------------------
1. Frontend (React/Vite): Deploy lên AWS S3 (Static Website Hosting) kết hợp với AWS CloudFront (CDN & HTTPS).
2. Backend (FastAPI): Deploy lên AWS EC2 (chạy Docker hoặc Nginx + Uvicorn) hoặc Serverless AWS Lambda + API Gateway.
3. Database: Di chuyển từ SQLite (database/app.db) sang AWS RDS (PostgreSQL hoặc MySQL).
4. File Storage: Di chuyển dữ liệu ảnh logs webcam và ảnh raw người dùng đăng ký lên AWS S3 Bucket.
5. Logging & Monitoring: Đẩy các log ứng dụng và lịch sử nhận diện về AWS CloudWatch.
